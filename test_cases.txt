نمونه‌های تست برای کتابخانه عملیات ماتریسی و برداری
==================================================

1. جمع ماتریس‌ها:
   A = [[1, 2], [3, 4]]
   B = [[5, 6], [7, 8]]
   نتیجه: [[6, 8], [10, 12]]

2. ضرب ماتریس‌ها:
   C = [[1, 2], [3, 4], [5, 6]] (3×2)
   D = [[1, 2, 3], [4, 5, 6]] (2×3)
   نتیجه: [[9, 12, 15], [19, 26, 33], [29, 40, 51]] (3×3)

3. دترمینان ماتریس:
   E = [[2, -1, 0], [-1, 2, -1], [0, -1, 2]]
   نتیجه: 4.0000

4. ماتریس معکوس:
   E = [[2, -1, 0], [-1, 2, -1], [0, -1, 2]]
   نتیجه: [[0.75, 0.50, 0.25], [0.50, 1.00, 0.50], [0.25, 0.50, 0.75]]

5. ضرب داخلی بردارها:
   v₁ = [1, 2, 3]
   v₂ = [4, 5, 6]
   نتیجه: 32.0000

6. ضرب خارجی بردارها:
   v₁ = [1, 2, 3]
   v₃ = [1, 0, -1]
   نتیجه: [-2.00, 4.00, -2.00]

7. تست خطا - ضرب خارجی با ابعاد نامعتبر:
   v₁ = [1, 2, 3] (3 بعدی)
   v₄ = [2, 3] (2 بعدی)
   نتیجه: خطا - ضرب خارجی فقط برای بردارهای سه‌بعدی تعریف شده است

8. اندازه و نرمال‌سازی:
   v₁ = [1, 2, 3]
   اندازه: 3.7417
   بردار نرمال‌شده: [0.2673, 0.5345, 0.8018]
